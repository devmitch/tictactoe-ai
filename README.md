# tictactoe-ai
Tic-Tac-Toe AI implemented with Minimax algorithm.

## requirements
- openjdk  10.0.1 (or later)
- boredom

## usage
In the git directory, run

`` javac -d ttt *.java``

``cd ttt/``

``java tictactoe.Menu``
